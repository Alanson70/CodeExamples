Требует файлов библиотек Function и Connection

Functions - библиотека реализующая дополнение среды С++Builderb,в том числе функциями API. 

Connections - исходные файлы библиотеки для организации доступа к серверу по интерфейсам TCP/UDP/IP и последовательным TTY/COM/RS232.

Спектроанализатор
В каталоне 2 проекта: SPECTROS_A - клиентская часть анализатора, встраиваемого в АСУ "РАССАДА";sanalyser - клиентское приложение, подключаемое  ксерверу АСУ "РАССАДА", позволяющее получать данные спектра, подготавливаемыеудаленным  сервером , соединенным со спектроанализатором произвольного типа.Также имеется возможность записывать и впоследствии воспроизводить запись изменения спектра.
