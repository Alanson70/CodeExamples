h1 Описание

Несколько клиенских проектов для ОС Windows, созданных в разные коды в среде разработке С++ Builder 6.0. 
Вместе с исходниками выложены исполняемые файлы , файлы настроек и сохранок, поскольку , во-первых, собрать проекты в современных версиях среды скорее всего будет проблематично. 
Во-вторых, цель представления этих проектов скорее показать интерфейсные решения нежели код, который, прямо сказать, здесь не блещет.
***

h1 Functions 

Файлы библиотеки (точнее подборки функций), реализующей дополнение среды С++Builderb,в том числе функциями API. 

h1 Connections 

Исходные файлы библиотеки для организации доступа к серверу по интерфейсам TCP/UDP/IP и последовательным TTY/COM/RS232. 
Библиотека широко применялась для создания клиентского ПО АСУ(автоматизированных систем управления). 
***

h1 TCP

Тестовая программа для отработки взаимодействия сервера и клиента на основе TServerSocket и TClientSocket. Применяется для отладки нижеследующих программ.

h1 UniTerm (2006)

Универсальный терминал - применялся при отладке взаимодействия с серверными устройствами разных типов, имеющих как сетевые интфесы TCP/UDP/IP, так и последовательный асинхронный интерфейс TTY(COM,RS232).
Для сборки требует файлов библиотек Function и Connection.

h1 SAnalyser (2016)

Клиентская часть анализатора спектра. 
Данные запрашиваются с сервера, осуществляющего измерения с помощью встроенного тюнера или сопряжение с анализаторами других типов, включаемых в состав станций спутниковой связи.
Позволяет записывать и воспроизводить изменение картины частотного спектра во времени. Примеры записей приведены в подкаталоге Records.
Для сборки требует файлов библиотек Function и Connection.
***

h1 EssToSat(2013)

Расчет направления и расстояния до спутника.
Упрощенная версия включена в сборник SatService.

h1 PXY(2013)

Расчет погрешности наведения для систем наведения под управлением АСУ "РАССАДА".
Упрощенная версия включена в сборник SatService.

h1 SatService (2003)

Набор утилит, объединенных в одной программе для быстрого запуска, необходимых в работе инженеров спутниковой связи, обслуживающих земные станции спутниковой связи (ЗСС).
Включает следующие расчеты:
* перевод децибелов в разы 
* расстояние от ЗСС до спутника
* направление до PCC до спутника
* расчет доплерова буфера для спутниковых модемов
* расчет погрешности наведения по параметрам ЗСС
* создание частотного плана сети

h1 ProgHelper (2010)

Попытка создать инструмент, позволяющий выполнять преобразования между различными видами представлений величин, дат, времени, цветов и т.п..

h1 Multiplication (2014)

Тренажёр таблицы умножения
